## 🔧 技术细节

### CSS 动画优化

**保留的动画：**
1. `bounce` - 标题图标跳动（轻量级）
2. `pulse` - 提示框呼吸效果（轻量级）
3. `spin` - 上传中旋转（功能性）

**移除的动画：**
1. ❌ `float` - 装饰图标浮动
2. ❌ `wave` - 波浪动画
3. ❌ `fadeInDown` - 淡入下降
4. ❌ `fadeInUp` - 淡入上升

**性能提升：**
- 减少 GPU 渲染压力
- 降低内存占用
- 提升页面流畅度

### 颜色系统

**主色调（金色系）：**
```css
/* 标题背景 */
background: linear-gradient(135deg, 
  rgba(255, 215, 0, 0.9),    /* 金色 */
  rgba(255, 165, 0, 0.85)    /* 橙色 */
);
```

**警告色（橙红系）：**
```css
/* 提示框 */
background: rgba(255, 87, 51, 0.15);  /* 淡橙红 */
border: 2rpx solid rgba(255, 87, 51, 0.3);
color: #ff5733;  /* 橙红色文字 */
```

**功能色（紫色系）：**
```css
/* 按钮和强调元素 */
background: linear-gradient(135deg, #667eea, #764ba2);
border: 3rpx dashed #667eea;
```

## 🎯 设计原则总结

### 1. 一致性优先
- 所有页面统一足球背景
- 所有页面标题统一金色底
- 统一的圆角和阴影系统

### 2. 信息层次
- 标题：最高层次（金色渐变 + 大字号）
- 警告：次高层次（橙红色 + 动画）
- 普通内容：基础层次（白底 + 灰文）

### 3. 克制的动画
- 仅在必要处使用动画
- 动画轻量化，不影响性能
- 功能性动画优先（如上传旋转）

### 4. 语义化颜色
- 金色 = 重要标题
- 橙红色 = 警告提示
- 紫色 = 操作按钮
- 白色 = 内容区域
