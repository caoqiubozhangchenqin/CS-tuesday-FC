# 🎯 在哪里输入代码 - 详细图文指南

## 📱 第1步：打开微信开发者工具

1. 在电脑上找到并打开 **"微信开发者工具"**
2. 打开您的小程序项目（CS-tuesday-FC）

---

## 📍 第2步：找到控制台（Console）

### 位置说明：
```
微信开发者工具窗口
├─ 顶部菜单栏（文件、编辑、工具...）
├─ 左侧文件列表（显示项目文件）
├─ 中间代码编辑区
├─ 右侧模拟器（显示小程序界面）
└─ 底部工具栏 ← 【这里！】
    ├─ Console（控制台）← 【在这个标签页】
    ├─ Sources
    ├─ Network
    ├─ AppData
    └─ Storage
```

### 具体操作：
1. 看微信开发者工具的**最底部**
2. 找到一排标签页：`Console`、`Sources`、`Network` 等
3. 点击 **`Console`** 标签（如果已经是 Console 就不用点）
4. 下方会出现一个**输入框**，旁边有个 `>` 符号

---

## ⌨️ 第3步：输入代码

### 在输入框中：
1. 将鼠标光标点击到 `>` 后面的空白处
2. 粘贴以下代码（可以直接复制多行）

```javascript
wx.cloud.database().collection('novels').add({
  data: {
    name: '我 中国队长',
    author: '未知作者',
    intro: '一本超过10MB的大型小说',
    category: '未分类',
    format: 'TXT',
    fileID: 'cloud://cloud1-3ge5gomsffe800a7.636c-cloud1-3ge5gomsffe800a7-1373366709/小说/我 中国队长.txt',
    cloudPath: '小说/我 中国队长.txt',
    size: 10485760,
    sizeText: '> 10 MB',
    uploadTime: Date.now()
  }
}).then(res => {
  console.log('✅ 添加成功！ID:', res._id);
  wx.showToast({ title: '已添加到书架', icon: 'success' });
});
```

3. 按 **Enter 键**执行

## 🎬 详细步骤动画说明

### 步骤1：找到底部区域
1. 眼睛看向微信开发者工具的**最底部**
2. 如果底部区域很小或看不到，可以**向上拖动边框**来放大

### 步骤2：切换到 Console
1. 看到底部的标签栏：`Console`、`Sources`、`Network`...
2. 点击 **`Console`** 标签
3. 下方会显示一个**深色或浅色的区域**

### 步骤3：找到输入框
1. 在 Console 区域的**最下方**
2. 有一个输入框，前面有个 **`>`** 符号
3. 这就是输入代码的地方！

### 步骤4：输入代码
1. 点击 `>` 后面的空白处
2. **Ctrl+V** 粘贴代码
3. 按 **Enter 键**
4. 看到输出结果

---

## ✅ 验证是否在正确位置

### 如果输入位置正确，会看到：
- ✅ 输入框前面有 `>` 符号
- ✅ 可以输入多行代码
- ✅ 按 Enter 后会立即执行并显示结果
- ✅ 上方显示之前的输出历史

### 如果输入错误的位置：
- ❌ 在代码编辑器中（.js 文件）← 这是写代码的地方，不是执行的地方
- ❌ 在 VS Code 终端中 ← 那是运行 PowerShell 的地方
- ❌ 在浏览器控制台中 ← 那是网页调试的地方

---

## 🆘 如果找不到 Console 标签

### 方法1：使用快捷键
按 **`Ctrl + Shift + C`**（Windows）或 **`Cmd + Shift + C`**（Mac）

### 方法2：通过菜单打开
1. 点击顶部菜单 **"调试"**
2. 选择 **"调试控制台"** 或 **"Console"**

### 方法3：调整窗口大小
1. 如果底部区域太小，用鼠标**拖动边框**向上拉
2. Console 区域会变大，更容易看到输入框

## 🎯 第一次使用的简单测试

在 Console 输入框中输入：

```javascript
console.log('Hello, 我找到控制台了！')
```

按 Enter，如果看到输出 `Hello, 我找到控制台了！`，说明您找对地方了！✅

## 💡 常见问题

### Q1: 我按了 Enter 但没反应？
**A**: 检查是否在 Console 标签页，不是在 Sources 或其他标签

### Q2: 提示语法错误？
**A**: 确保复制了完整的代码，包括所有的 `{` `}` 和 `;`

### Q3: 提示"wx is not defined"？
**A**: 说明不在微信开发者工具中，请检查是否在正确的工具中

### Q4: Console 区域显示很多其他信息？
**A**: 那些是正常的日志，忽略它们，直接在输入框输入代码即可
