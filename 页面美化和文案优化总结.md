## 🎨 设计细节

### 动画效果

#### 1. 淡入动画（fadeInDown）
```css
@keyframes fadeInDown {
  from {
    opacity: 0;
    transform: translateY(-30rpx);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```
**应用于：** 页面标题
**效果：** 从上方淡入

#### 2. 淡入上移（fadeInUp）
```css
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(30rpx);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```
**应用于：** 表单、使用提示
**效果：** 从下方淡入，错开时间（0.2s、0.4s）

#### 3. 浮动动画（float）
```css
@keyframes float {
  0%, 100% { transform: translateY(0) rotate(0deg); }
  50% { transform: translateY(-30rpx) rotate(10deg); }
}
```
**应用于：** 顶部装饰图标
**效果：** 上下浮动并轻微旋转

#### 4. 波浪动画（wave）
```css
@keyframes wave {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-10rpx); }
}
```
**应用于：** 底部装饰
**效果：** 轮流上下移动

#### 5. 旋转动画（spin）
```css
@keyframes spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}
```
**应用于：** 上传中的 ⏳ 图标
**效果：** 持续旋转

### 交互反馈

#### 按钮点击效果
```css
.choose-btn:active,
.upload-btn:active {
  transform: scale(0.98);  /* 缩小到 98% */

```

#### 卡片点击效果


## 🎯 设计理念

### 1. 渐进式动画
- 不同元素按顺序淡入（0s → 0.2s → 0.4s）
- 创造层次感和节奏感

### 2. 微交互反馈
- 每个可点击元素都有视觉反馈
- 增强用户操作确定性

### 3. 装饰性元素
- 不干扰主要内容
- 半透明、轻微动画
- 提升页面活力

### 4. 一致性
- 统一的紫色系主题
- 统一的圆角（20rpx）
- 统一的阴影效果
