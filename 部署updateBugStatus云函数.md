# 部署updateBugStatus云函数

## 修改内容
新增了 `cloudfunctions/updateBugStatus` 云函数，用于管理员更新bug状态。

## 部署步骤

### 方法1：在微信开发者工具中部署

1. 打开微信开发者工具
2. 在左侧目录树中找到 `cloudfunctions/updateBugStatus` 文件夹
3. 右键点击 `updateBugStatus` 文件夹
4. 选择 **"上传并部署：云端安装依赖"**
5. 等待部署完成（通常需要几秒钟）

### 方法2：使用命令行部署

在项目根目录下运行：

```bash
# 进入云函数目录
cd cloudfunctions/updateBugStatus

# 部署云函数
wx-server-sdk deploy
```

## 验证

部署完成后：
1. 重新编译小程序
2. 以管理员身份进入bug提交页面
3. 点击bug卡片底部的操作按钮（🔄 处理中、✅ 已解决、🔙 重新打开）
4. 应该能成功更新bug状态

## 如果部署失败

可能的原因：
1. 网络问题 - 检查网络连接
2. 权限问题 - 确保有云开发权限
3. 环境配置问题 - 检查云开发环境是否正确配置

## 调试

在小程序控制台查看错误信息，或者在云开发控制台查看云函数日志。
